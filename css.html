CSS: Styling language

   How to insert a style sheet:

  • External CSS:
    • <head>
         <link rel="stylesheet" href="mystyle.css">
    </head>
  • Internal CSS:
    • <head>
         <style>
              body {background-color: powderblue;}
         </style>
    </head>
  • Inline CSS:
    • <h1 style="color:blue;">A Blue Heading</h1>
